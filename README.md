# Emergency Disaster Response Admin Dashboard

The **Emergency Disaster Response Admin Dashboard** is a web-based control system designed to help administrators manage, monitor, and coordinate disaster-related alerts and emergency responses in real time.

It supports emergency reporting, user monitoring, and announcement management for faster and more organized disaster response operations.

---

## 🚨 Purpose

This system is built to improve disaster preparedness and response by allowing administrators to:

- Monitor incoming emergency reports (e.g., fire, flood, landslide)
- Send real-time alerts and announcements
- Manage user-submitted incident reports
- Coordinate response actions efficiently

---

## ✨ Features

- 🔐 Secure admin login system
- 📢 Post and manage emergency announcements
- 🚨 View and track user-reported incidents
- 📍 Monitor disaster categories (Fire, Flood, Landslide, etc.)
- 📊 Dashboard overview for quick situational awareness
- 🧑‍💻 User management (if enabled)
- 📷 Support for image/video evidence from users

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Database:** Firebase (depending on configuration)  
- **Tools:** Git, VS Code
